# \# Secure File Sharing

# 

# A secure cloud file sharing web application that stores files privately in AWS S3 and generates temporary presigned URLs for controlled downloads.

# 

# \## Project Goal

# 

# The goal of this project is to build a cloud-based file sharing platform where users can upload files securely, keep them private in AWS S3, and share them using time-limited download links.

# 

# \## Planned Features

# 

# \- Upload files through a web interface

# \- Store files in a private AWS S3 bucket

# \- Generate temporary presigned download URLs

# \- Allow users to choose link expiry time

# \- Display upload status and download link

# \- Keep AWS credentials hidden on the backend

# \- Add file history/dashboard later

# \- Add authentication later

# 

# \## Tech Stack

# 

# \- React

# \- Node.js

# \- Express

# \- AWS S3

# \- AWS SDK for JavaScript

# \- GitHub
